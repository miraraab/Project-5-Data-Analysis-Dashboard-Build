# Timeline Estimate

## Phase 0: Discovery (Weeks 1–4)
- KPI inventory across departments
- Data source audit
- Stakeholder alignment
- Tool selection decision
- Deliverable: Go/No-Go for pilot

## Phase 1: Pilot (Months 2–4)
- Tool setup and configuration
- Data integration (1 department)
- Dashboard build
- User training (pilot group)
- Deliverable: Pilot dashboard live, feedback collected

## Phase 2: Evaluation (Month 5)
- Measure against success metrics
- Collect qualitative feedback
- Decision: expand, adjust, or stop
- Deliverable: Pilot evaluation report + rollout recommendation

## Phase 3: Rollout (Months 6–18)
- Department-by-department expansion
- Full integration with existing reporting cycles
- Change management program
- Deliverable: Organization-wide strategy monitoring live

## Key Dependencies
- IT sign-off on data access (BSI compliance check)
- Department head buy-in before pilot launch
- Budget approval for rollout (separate from pilot budget)
