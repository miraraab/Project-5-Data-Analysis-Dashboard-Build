# Cost Analysis

## Pilot Phase (3 months)
| Item | Cost Estimate | Assumption |
|---|---|---|
| Tool licensing (Perdoo or similar) | €2,000–5,000 | 20 users, 3 months |
| Internal project lead (0.5 FTE) | €10,000–15,000 | Mid-level, 3 months |
| Data preparation / cleaning | €5,000–8,000 | External support or intern |
| Change management / training | €2,000–3,000 | 2 workshops |
| **Pilot Total** | **€19,000–31,000** | |

## Rollout Phase (12 months post-pilot)
| Item | Cost Estimate | Assumption |
|---|---|---|
| Tool licensing (full org) | €15,000–30,000/yr | 100 users |
| Implementation support | €30,000–50,000 | External consultant |
| Internal PM (1.0 FTE) | €60,000–80,000 | Senior level |
| IT integration | €20,000–40,000 | Legacy system connectors |
| **Rollout Total** | **€125,000–200,000** | Year 1 only |

## ROI Estimate
- Manual reporting effort saved: ~2h/week per department head × 15 dept heads = 30h/week
- At €100/h fully loaded cost = €3,000/week = €150,000/year
- Break-even: ~12-16 months post-rollout

*Assumption (medium confidence): 2h/week is conservative estimate based on Deloitte public sector benchmarks*
